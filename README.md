# My neovim config

**NOTE:** This is for AstroNvim v4+


## 🛠️ Installation

What this install requires:

- ripgrep
- nvim
- rust
- treesitter-cli
- graphviz
- fzf
- lldb

Or, you can just use my autoinstall script found in my dotfiles repo

#### Clone the repository

If you have the requirements met, go ahead and clone the repo

```shell
git clone https://github.com/Jstone751/nvim ~/.config/nvim
```

#### Start Neovim

```shell
nvim
```
Enjoy! :)
